# WB KBT Analyze

Tampermonkey userscript для отчета по завершенным рейсам на `https://logistics.wildberries.ru/*`.

## Установка

Откройте raw-файл в браузере:

https://raw.githubusercontent.com/AleksTatarinov/WB_KBT_Analize/main/dist/wb-report.user.js

Tampermonkey предложит установить скрипт. Обновления будут подтягиваться по `@updateURL`.
